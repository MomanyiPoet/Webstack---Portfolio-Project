This project is about:
## POETIC INSPIRATION BLOG WEBSITE
- This website is a collection of poetry resources designed to inspire and engage readers.
The website features four pages:

## 1. Home page:
- This page provides an introduction to the blog website and the content that can be found on the other pages.

## 2. Login page: 
This page allows users to keyy in their details and sign in.

## 3. Dashboard page:
This page allows users to write their blogs and publish their content.
